A box-model lényegében egy doboz ami minden html elemet körül ölel. Ez a doboz a tényleges tartalomból, paddingből, borderből, és marginből állhat. A fent látható elem is egy ilyen doboz, aminek tényleges tartalma nincs, helyette megadtam a méreteit: 

            
                div{
                    background: lightcoral;
                    height: 300px;
                    width: 300px;
                    padding: 30px;
                    margin: 50px auto;
                    border: 2px solid red;
                    box-sizing: border-box;
                }
A box-sizing: border-box; miatt a belső tér a padding és a border eggyüttese teszi ki a 300px-t! A margin: 50px auto; miatt azonban a teljes box-modellunk magassága 400px ((2 * 50px) + 300px), szélessége pedig megegyezik a teljes oldal szélességgel (auto)
